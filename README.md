1. <!-------  var, let, const ------->
    var - old way of declaring variables, works in whole function, can be reassigned.
    let - works only in the block(block-scoped), can be reassigned.
    const - block-scoped, cannot be reassigned.

2. <!------- spread operator ------->
    Spread Operator expands an array or an object into individual items.

3. <!------- map(), filter(), forEach() -------->
    --> map() -- returns a new array after transforming each element in the original array.
    --> filter() -- returns a new array containing only elements that follow given condition.
    --> forEach() -- loops through elements but does not return any new array, returns undefined.

4. <!------- arrow function ------->
    Arrow function is a shorter way to write a function.

5. <!------- template literals ------->
    Template literals is using backticks (`) to easily include variables or expressions in strings and to write multiline code.
